Hello, Datamart Users!

This Datamart is designed to combine data from multiple analysis pipelines into one unified interface. All data should have gone through quality assurance already so that everything in the datamart can be treated as good. (Nonetheless, you'll be able to flag data as bad.)

Complex Filtering

Using the web interface or the REST API you can construct complex queries of the data and export it in a format compatible with your favorite tools.

Uses LDAP

The next release will support authenticating via LDAP so you don't have to manage another set of login credentials.

All your data

Data is only usefull if you can query it how you want. Combine any data from any source to your heart's content!
